# CodeSync — Real-time Collaborative Code Editor

A real-time collaborative code editor built with React and WebSockets.
Built as a Google Internship portfolio project.

## Features
- Real-time code sync across multiple users
- Syntax highlighting for 8 languages (JS, Python, Java, C++, TS, HTML, CSS, JSON)
- Live code execution (JavaScript runs in browser)
- Multi-user rooms with shareable Room ID
- Live chat panel
- Online user presence

## Tech Stack
- **Frontend:** React 18 + Vite
- **Real-time:** WebSocket broadcasting (in-memory for demo)
- **Styling:** CSS-in-JS

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open http://localhost:5173 in your browser.

## Deploy to Vercel
1. Push to GitHub
2. Import repo on vercel.com
3. Click Deploy
